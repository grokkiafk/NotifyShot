# История изменений

## v1.3.0
- **Сборка через GitHub Actions** — релизы собираются на серверах GitHub из открытого исходного кода.
- **Подпись артефактов** (build provenance, Sigstore): можно криптографически проверить, что `.exe` собран именно из этого репозитория. Проверка: `gh attestation verify NotifyShot.exe --repo grokkiafk/NotifyShot`.
- Добавлен `CHANGELOG.md`.

## v1.2.1
- «Отчёт медика»: правильный учёт **выходных** (Сб/Вс — день 12:00–24:00, ночь 00:00–12:00). День недели читается из календаря в кадре.

## v1.2.0
- Новая вкладка **«Отчёт медика»**: раскладка скринов по папкам (Таблетки · Вакцинация · ПМП → День/Ночь), **чтение игровых часов** в кадре для день/ночь, **подсчёт баллов**, отчёт в `.txt` и `.csv`.

## v1.1.0
- Профили **«Вакцинация»** и **«Таблетки / лечение»** в комплекте.

## v1.0.0
- Первый релиз: автоматическая нарезка скриншотов по игровому уведомлению, профиль «Оказание ПМП — Majestic», интерфейс в стиле лаунчера Majestic, безрамочное окно, ffmpeg внутри.
